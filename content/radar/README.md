# Content radar

Daily inbound content ideas, gathered by the content-radar cloud routine each morning
(~7:15am Lagos). Each day lands as `YYYY-MM-DD.md` here and as a pull request titled
"Content radar — <date>", so you get a GitHub notification you can read from your phone.

Each item has: a one-line hook in your voice, 2-3 sentences of context, your angle, a
suggested format (LinkedIn post / article / quick note), and a source link.

The 2-3 strongest picks each day are also appended to `../backlog.md`, so you can ask
Claude to "draft the blog post about X" and the content-engine skill picks it up.

Nothing here is auto-published. You read it, pick what is worth your voice, and ignore the
rest.
